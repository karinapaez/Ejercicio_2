Presentación sobre reutilización de software

El proceso de industrialización del software, exige que los ingenieros y técnicos planteen nuevas alternativas para incrementar la productividad de los desarrolladores y analistas en el desarrollo de sistemas de software.

Se define la reutilización de software como el “uso de cualquier tipo de artefacto (también llamado activo), o parte del mismo, creado con anterioridad, en un nuevo proyecto.”

Conclusión: La reutilización de software aparece como una alternativa para desarrollar aplicaciones y sistemas SW de un manera más eficiente, productiva y rápida. La idea es reutilizar elementos y componentes SW en lugar de tener que desarrollarlos desde un principio.
